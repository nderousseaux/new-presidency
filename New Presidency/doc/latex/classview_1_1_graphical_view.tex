\doxysubsection{Detailed Description}
{\bfseries{{\itshape \mbox{\hyperlink{classview_1_1_graphical_view}{Graphical\+View}}} est la classe de l\textquotesingle{}interface graphique du jeu}} 

L\textquotesingle{}interface est une J\+Frame, comportant d\textquotesingle{}autres éléments\+: 
\begin{DoxyItemize}
\item des {\bfseries{J\+Pannels}} pour les indicateurs, les leviers, le budget et le tour 
\item des {\bfseries{J\+Scroll\+Pane}} permettant d\textquotesingle{}avoir un ascenceur vertical 
\item des {\bfseries{J\+Button}} pour afficher les graphiques d\textquotesingle{}évolution et terminer le tour 
\end{DoxyItemize}Le côté flexible de l\textquotesingle{}interface est amené par le combo de Layout {\bfseries{Border\+Layout}} pour la fenêtre principale, et {\bfseries{Grid\+Layout}} pour le pannel comprenant les indicateurs et les leviers, ainsi que le pannel au bas de l\textquotesingle{}interface, comprenant le budget et les boutons.
