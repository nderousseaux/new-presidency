\doxysubsection{Detailed Description}
{\bfseries{{\itshape \mbox{\hyperlink{classmodel_1_1_scenario}{Scenario}}} est la classe de chacun des scénarios proposés dans le jeu}} 

Elle contient, pour chacune de ses instances\+: 
\begin{DoxyItemize}
\item Un {\bfseries{nom}} de scénario 
\item Une {\bfseries{abréviation}} du nom de scénario 
\item Une {\bfseries{situation de départ}} du scénario sous la forme d\textquotesingle{}une {\bfseries{Hash\+Map$<$String,Double}} 
\item Une {\bfseries{situation de victoire}} du scénario sous la forme d\textquotesingle{}une {\bfseries{Hash\+Map$<$String,Double}} 
\item Une {\bfseries{situation de défaite}} du scénario sous la forme d\textquotesingle{}une {\bfseries{Hash\+Map$<$String,Double}} 
\item Une {\bfseries{liste d\textquotesingle{}informations}} relative au scénario, affichée à l\textquotesingle{}utilisateur pour lui indiquer comment jouer 
\end{DoxyItemize}
